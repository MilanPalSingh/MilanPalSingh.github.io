Deploying Hybrid App on an iPhone

Steps to follow:

  1. Set Up Ionic
  2. Build App for IOS
  3. Open App in Xcode
  4. Request your certificates
  5. Configure the device for development
  6. Create a provisioning profile for your app
  7. Make sure your app is set to use that Xcode provisioning profile
  8. Finally deploy the app to the device
